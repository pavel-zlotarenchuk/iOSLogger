## Installation

iOSLogger is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
use_frameworks!
pod 'iOSLogger', :git => 'https://github.com/pavel-zlotarenchuk/iOSLogger.git'
```
And run  `pod install`
